# OCR-licencia-de-placas
Este proyecto fue desarrolla por estudiantes de la UNIVERSIDAD NACIONAL DE Ingeniería del curso de ELECTRONICA APLICADA 
TIENE COMO OBJETIVO Los sistemas ANPR integrados con YOLO11 son una gran opción para gestionar el acceso a zonas seguras como comunidades cerradas, campus corporativos e instalaciones restringidas. Mediante el uso de ANPR, estos lugares pueden automatizar su seguridad, asegurándose de que sólo se permite la entrada a los vehículos autorizados.
